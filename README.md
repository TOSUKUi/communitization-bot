## Abstract

convert your discord text to what stalin will talking like.

ディスコードのテキストをスターリンが話すように修正します。

Japanese only now.

## How to use(使い方)
1. set these environment variables.
```
DISCORD_BOT_TOKEN
OPENAPI_API_KEY
```

2.  install and run
```
git clone https://github.com/TOSUKUi/communitization-bot
cd communitization-bot
poetry install
poetry shell
python main.py
```
